## 파일 설명
- thyme : IoT 데이터를 관리하기 위한 다양한 통신 프로토콜을 설정하고, 데이터를 파일로 저장하거나, 특정 경로에 따라 다르게 처리하며, Mobius 플랫폼과의 통신을 담당하는 주요 기능을 포함
- tas :  IoT 시스템에서 conf.xml에 저장된 설정값을 읽어와 TAS 상태에 따라 데이터를 송수신하고, 제어 신호를 전달하는 역할
